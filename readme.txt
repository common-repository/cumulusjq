== Description ==

wordpress plugin that animates links or category links to form a sphere, uses jquery framework, if bug exist please report.

== Note ==
'DO NOT USE THE PLuGIN UNDER A ("STYLE" POSITION ABSOLUTE OR POSITION RELATIVE PARENT TAG IN THE FRONT-END)'

== Installation ==

1. Upload `cumulusjq` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
